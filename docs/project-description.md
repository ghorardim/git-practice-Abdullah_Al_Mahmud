# Project Description

## Overview

This project is created as part of a Git and GitHub practice exercise.  
The main goal of this project is to understand and apply basic version control concepts using Git and remote repository management using GitHub.

---

## Objectives

- Practice creating a local Git repository
- Learn how to stage and commit files
- Understand how to connect a local repository to a remote GitHub repository
- Practice writing meaningful commit messages
- Use a proper project folder structure

---

## Project Structure

The project follows a simple and organized structure:

- README.md → General project information
- .gitignore → Files and folders to ignore in version control
- src/ → Source code directory
    - main.py → Main program file
    - utils.py → Utility/helper functions
- docs/ → Documentation folder
    - project-description.md → Detailed project explanation

---

## Features

Currently, the project includes:

- Prints the developer’s name
- Prints the current date
- Print the addition, subtraction, and multiplication of two numbers.
- Clean and organized directory structure
- Basic Git version control setup

---

## Technologies Used

- Python
- Git
- GitHub

---

## Learning Outcome

By completing this project, the developer gains hands-on experience with:

- Initializing a Git repository
- Creating commits
- Using .gitignore correctly
- Managing project structure
- Pushing code to GitHub
